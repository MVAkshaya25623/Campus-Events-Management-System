# Campus Events Management System

## Overview

Campus Events Management System is a full-stack web application developed using Spring Boot that helps manage campus events, user registrations, and event participation. The system provides secure authentication and enables efficient event organization and tracking.

## Features

* User Registration and Login
* Secure Authentication using Spring Security
* Event Creation and Management
* Event Registration for Students
* Event Participation Tracking
* Responsive User Interface
* Database Integration

## Technologies Used

* Java
* Spring Boot
* Spring Security
* Thymeleaf
* HTML
* CSS
* JavaScript
* MySQL
* Maven

## Project Structure

* Controller Layer
* Service Layer
* Repository Layer
* DTO Layer
* Security Layer
* Model Layer

## Installation

1. Clone the repository.
2. Configure the database in `application.properties`.
3. Run the project using Maven.
4. Access the application through your browser.
